# CV Builder Web App

## Overview
CV Builder is a web application that allows users to create, edit, and download professional CVs in PDF format.

The application includes user authentication, data storage, and a dashboard where users can manage their CVs.

---

## Tech Stack
- Frontend: React, JavaScript, Bootstrap
- Backend / Database: Firebase (Authentication, Firestore / Realtime DB)
- Design: Figma (UI/UX, prototyping)
- PDF Generation: JavaScript PDF library

---

## Features
- User registration and login (Firebase Authentication)
- Dynamic forms for CV creation
- CV generation in PDF format
- Download CV as PDF
- User dashboard
- Edit and delete CV functionality
- Data storage using Firebase database

---

## Architecture
- React frontend handles UI and logic  
- Firebase provides backend services (authentication + database)  
- Client communicates directly with Firebase APIs  

---

## Setup

### Frontend
- Install dependencies:
  npm install
- Run the app:
  npm start

### Firebase
- Create Firebase project
- Enable Authentication
- Set up Firestore / Realtime Database
- Add Firebase config to the project
